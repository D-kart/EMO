## What is EMO.AI

EMO.AI 是一个面向**个人投资者、上市公司高管、企业家**三类决策者的 AI 投研工具系列。

每一个工具，回答这三类用户每天都在问、却很少被认真回答的一个具体问题。

我们做的，是把全球公开披露的金融数据（SEC EDGAR、港交所披露易、巨潮资讯）做成可用的研究产物：研报、持仓追踪、并购扫描、市值管理决策支持。

## Data Sources

EMO.AI 全部数据来自交易所与监管机构的**公开披露文件**，无任何非公开渠道：

- **SEC EDGAR** — 13F / 13D / 13G / 中概 ADR 持仓
- **HKEX 披露易** — 大股东权益变动 / CCASS 持股
- **巨潮资讯网** — A 股定期报告 / 前十大流通股东
- **沪深交易所** — 龙虎榜 / 机构席位
- **基金业协会** — 公募基金披露
- **Futu OpenAPI** — 行情底图与 K 线渲染
